## 🔁 照応とは何か？

> **照応とは、情報・意思・縁（えにし）が、時間・空間・因果律の中で移動・拡散・共振・共鳴・接続・ハンドシェイクする現象である。**

その背後にある意味の背景圧・文脈の密度・構造の共鳴強度こそが、照応の深度（Depth of Correspondence）を決定します。
